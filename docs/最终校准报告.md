## 🚀 推荐运行命令

使用以下命令可在软件中复刻硬件设备的听感：

```powershell
python process_tomatis.py `
  -i "输入文件.flac" `
  -o "输出文件.flac" `
  --gate_ui 50 `
  --gate_scale 1.0 `
  --gate_offset -74.3 `
  --hyst_db 3 `
  --up_delay_ms 0 `
  --fc 1000 `
  --slope 12 `
  --c1_low 5 --c1_high -5 `
  --c2_low -5 --c2_high 5 `
  --n_fft 4096 --hop 2048 `
  --state_csv "state.csv"
```
